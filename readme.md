# Notas
Este es un programa sencillo en Node para multiplicar
```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     El número que queremos multiplicar         [número] [requerido]
  -h, --hasta    El número hasta el cual queremos multiplicar
                                                          [número] [defecto: 10]
  -l, --listar   Listar o no la tabla de multiplicar [booleano] [defecto: false]
```
